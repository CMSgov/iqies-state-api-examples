# JavaScript(Node) example for iQIES State API

Simple Node Client (requests-driven) script example for iQIES State API.

## Getting Started

In this example, we're reading our query from a file to make it easier to tweak and output results in json and csv format.

### Built with: 

* [npm](https://www.npmjs.com/) - npm is the package installer for Node
* [node](https://nodejs.org/en/download/) - Node installation

### Prerequisites

- [ ] To use this script you need to install the libraries above. Please refer the documentation from links provided above.

### Setting the environment variables ###

Refer: [How to set environment variables](../docs/set_env_variables.md)

### Running this script

- Clone the code into your local working directory.
- Navigate to javascript_example directory.

	`cd <local working directory>/iqies-state-api-examples/javascript_example`

- Run `npm -s start` to execute the program and generate results. 
- The generated output files in both json and csv format are placed under same folder where the program is executed from.
