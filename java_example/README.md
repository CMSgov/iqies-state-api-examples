# Project Title

Simple Java Client (requests-driven) script example for iQIES State API.

## Getting Started

In this example, we're reading our query from a file to make it easier to tweak.

### Built with: 

* [Java 8](https://www.oracle.com/java/technologies/javase-downloads.html) - Java 8 or above
* [Spring boot](https://spring.io/projects/spring-boot) - This project is built with spring boot application framework.

### Prerequisites

- [ ] To run the java application, install Java 8 or above.

### Setting the environment variables ###

Refer: [How to set environment variables](../docs/set_env_variables.md)

### Running this script

- Clone the code into your local working directory.
- Navigate to java_example directory.

	`cd <local working directory>/iqies-state-api-examples/java_example`
	
- Run the command `./gradlew run` to execute the program and generate results. 
- The generated output files in both json and csv format are placed under same folder where the program is executed from.
