# Project Title

Simple Python 3.8.0 Client (requests-driven) script example for iQIES State API.

## Getting Started

In this example, we're reading our query from a file to make it easier to tweak.

### Built with: 

* [pip](https://pypi.org/project/pip/) - pip is the package installer for Python
* [python](https://www.python.org/downloads/) - Python 3.8.x or above

### Prerequisites

- [ ] To use this script you need to install the libraries above. Please refer the documentation from links provided above.

### Setting the environment variables ###

Refer: [How to set environment variables](../docs/set_env_variables.md)

### Running this script

- Clone the code into your local working directory.
- Navigate to python_example directory.

	`cd <local working directory>/iqies-state-api-examples/python_example`
	
- Run the command `pip3 install -r requirements.txt` to install dependencies.
- Run the command `python3 run_api_call.py` to execute the program and generate results. 
- The generated output files in both json and csv format are placed under same folder where the program is executed from.
